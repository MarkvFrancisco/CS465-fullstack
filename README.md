# Functionality

## How is JSON different from Javascript and how does JSON tie together the frontend and backend development pieces?
### JSON is different from Javascript in the way that JSON is responsible for exchanging information between the frontend and backend in a specific format, while Javascript is responsible for handling logic and interactivity within the frontend. JSON ties together the frontend and the backend development pieces through data communication between the frontend, then the server (Express), and then the backend (MongoDB).
